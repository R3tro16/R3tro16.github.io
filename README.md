# RETRO16 — Cyber Portfolio

Portfolio personnel sous forme d'OS rétro (boot MS-DOS → Windows 3.1).
Writeups CTF, bug bounty, game hacking.

🌐 https://r3tro16.github.io

## Stack

HTML / CSS / JS pur. Aucune build step. Hébergé sur GitHub Pages.

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```sh
python -m http.server 8000
```

## Ajouter un writeup

Dupliquer un `<li>` dans la fenêtre concernée (`index.html`).
Pour aller plus loin (markdown, articles dédiés), migration vers Astro prévue.
